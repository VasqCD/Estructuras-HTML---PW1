Ejercio 7 JavaScript

Realice una página web que tenga disponible una etiqueta Input de tipo text como “edad”, además, un botón llamado “evaluar”, en donde, una vez que el usuario escriba un valor numérico en el input y se presione el botón mediante un clic, este valor ingresado deberá condicionarse mediante una función JS, evaluando si esa edad es mayor o igual a 21 o menor de esta, mostrar un Alert con el resultado.
