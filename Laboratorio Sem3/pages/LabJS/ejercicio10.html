Ejercio 10 JavaScript

Crear una página web que utilice una función JS para llenar una tabla de datos (etiqueta HTML) de empleados, considere que la tabla debe contener datos como el número del empleado, el nombre, apellido, edad y sueldo. Haga uso de tabla HTML, botones, eventos, funciones JS, getElement del document y de InnerHTML.
